Web - Admin Dashboard

Admin Dashboard ini adalah aplikasi web berbasis React yang digunakan untuk menampilkan dan mengelola data pengguna melalui dashboard yang interaktif. Aplikasi ini mengambil data dari REST API dan memungkinkan pengguna untuk menambahkan serta menghapus data pengguna. Data tambahan yang dibuat oleh pengguna disimpan menggunakan localStorage agar tetap tersedia setelah halaman direfresh. Tampilan antarmuka dibangun menggunakan Tailwind CSS dan DaisyUI untuk menghasilkan UI dashboard yang modern, responsif, dan mudah digunakan.

Tech Stack: React, JavaScript, Tailwind CSS, DaisyUI, REST API, LocalStorage
